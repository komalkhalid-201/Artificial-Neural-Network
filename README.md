# Artificial-Neural-Network
ANN stands for Artificial Neural Network. It is a machine learning model inspired by the way the human brain processes information through interconnected neurons.
How an ANN works:
Input Layer – Receives the input data (e.g., age, salary, image pixels, etc.).
Hidden Layer(s) – Processes the information by performing calculations and finding patterns.
Output Layer – Produces the final prediction or result.

For example, if an ANN is trained to identify cats in images:

The input layer receives the image pixels.
The hidden layers learn features such as edges, ears, eyes, and whiskers.
The output layer predicts whether the image contains a cat or not.
Simple Structure:
Input Layer → Hidden Layer(s) → Output Layer
Common applications of ANN:
Image recognition
Speech recognition
Natural language processing
Medical diagnosis
Stock price prediction
Recommendation systems (Netflix, YouTube, Amazon)
Example:

Suppose you want to predict whether a student will pass an exam based on:

Study hours
Attendance percentage
Previous grades

These values are fed into the ANN, which learns patterns from previous students' data and predicts whether the student is likely to pass or fail.

ANN is the foundation of Deep Learning. When an ANN contains many hidden layers, it is called a Deep Neural Network (DNN).
